# Welcome to BenTelega

This is my personal knowledge base and documentation site, built with [Jekyll](https://jekyllrb.com/) and [Obsidian](https://obsidian.md/).

## About This Site

BenTelega serves as a central repository for my notes, documentation, and knowledge management system. It combines the power of Obsidian for local note-taking with Jekyll for publishing content to the web.

## Features

- **Knowledge Management**: Organize and structure my knowledge
- **Documentation**: Publish technical documentation and guides
- **Searchable Content**: Find information quickly
- **Version Control**: Track changes with Git
- **GitHub Pages**: Hosted on GitHub Pages
