# TMUX Настройки

## Быстрый старт

### Запуск TMUX
```bash
tmux new -s <имя_сессии>
```

### Подключение к существующей сессии
```bash
tmux attach -t <имя_сессии>
```

### Список сессий
```bash
tmux ls
```

## Горячие клавиши

- **Ctrl-a** - Префикс-клавиша (вместо стандартного Ctrl-b)
- **Ctrl-a c** - Создать новую окно
- **Ctrl-a %** - Разделить панель вертикально
- **Ctrl-a "** - Разделить панель горизонтально
- **Ctrl-a <arrow>** - Переключаться между панелями
- **Ctrl-a d** - Отключиться от сессии (оставить её работающей)
- **Ctrl-a I** - Установить/обновить плагины (только после загрузки конфигурации)
- **Ctrl-a T** - Выбрать тему (переключение между доступными темами)

## Плагины

### Установленные плагины:

1. **tmux-sensible** - Оптимальные настройки по умолчанию
2. **tmux-resurrect** - Сохранение и восстановление сессий
3. **tmux-continuum** - Автоматическое сохранение сессий
4. **tmux-copycat** - Улучшенное копирование текста
5. **tmux-prefix-highlight** - Подсветка префикс-клавиши

### Использование tmux-resurrect и tmux-continuum

- **Сохранить сессию**: Ctrl-a Ctrl-s
- **Восстановить сессию**: Ctrl-a Ctrl-r
- **Автоматическое сохранение**: Включено по умолчанию каждые 15 минут

### Использование tmux-copycat

- **Копировать текст**: Выделите текст мышью и нажмите Enter
- **Режим копирования**: Ctrl-a [

## Статус бар

Строка статуса отображает:
- **Левая часть**: Имя сессии и текущая тема
- **Правая часть**: Время, дата, текущая команда
- **Текущее окно**: Подсвечено в соответствии с выбранной темой

Доступные темы:
- `dark` - Тёмная тема (по умолчанию)
- `light` - Светлая тема
- `solarized-dark` - Solarized тёмная
- `solarized-light` - Solarized светлая
- `high-contrast` - Высокая контрастность

## Настройка

Конфигурационный файл: `~/.tmux.conf`

Для применения изменений:
```bash
tmux source-file ~/.tmux.conf
```

### Переключение тем

1. Нажмите **Ctrl-a T** для открытия меню выбора темы
2. Выберите одну из доступных тем
3. Тема применится сразу

### Сохранение предпочитаемой темы

Текущая тема сохраняется в переменной `@themepack-theme` и будет использоваться при следующем запуске tmux.

## Дополнительные команды

- **Переименовать сессию**: `tmux rename-session <новое_имя>`
- **Удалить сессию**: `tmux kill-session -t <имя_сессии>`
- **Переключиться между окнами**: `Ctrl-a <number>`
